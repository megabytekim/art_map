# UX Improvements Design

**Date:** 2026-03-08
**Target:** 지인들끼리 보는 용도

## 1. 마감 임박 배지 (D-14)
- endDate 기준 14일 이내면 목록/카드에 `마감 임박` 빨간 배지 표시
- 기존 블로그 인기순 정렬 유지

## 2. 지도 클러스터링
- react-leaflet-markercluster로 줌 아웃 시 숫자 묶음
- 줌 인하면 개별 CircleMarker 표시

## 3. 전시 상세 슬라이드업 카드
- 전시 선택 시 지도 아래에서 카드 올라옴
- 전시 기본 정보 + 네이버 블로그 상위 3~5개 (썸네일, 제목, 요약)
- 블로그 글 클릭 시 새 탭
- 기존 Popup 대체
- "네이버에서 더 보기" 링크 포함

## 4. 새로 만들 것
- `GET /api/blog-search` - 전시명+갤러리명으로 블로그 상위 결과 반환
- `SlideUpCard` 컴포넌트
- 마감 임박 유틸 함수

## 안 하는 것
- 정렬 옵션 변경, 갤러리 사이트 크롤링, 카테고리 필터
